# 凸优化教程大纲

## 课程目标
本课程旨在为机器人专业学生提供凸优化的理论基础和实用算法，重点介绍多目标优化和流形优化的核心概念及其在机器人领域的应用。通过本课程学习，学生将能够：
1. 掌握凸优化的基本理论和方法
2. 理解多目标优化的帕累托最优性概念
3. 熟悉流形优化的基本原理和算法
4. 能够将凸优化方法应用于机器人路径规划、控制和决策问题

## 课程内容

### 第一部分：凸优化基础（16学时）

#### 第1章：凸集与凸函数（4学时）
1.1 凸集的定义和基本性质
- 凸集的定义和判定条件
- 常见凸集：超平面、半空间、球、多面体等
- 凸集的运算：交集、仿射变换、透视变换等

1.2 凸函数的定义和性质
- 凸函数的定义和一阶/二阶判定条件
- 常见凸函数：线性函数、二次函数、指数函数等
- 凸函数的运算：非负加权和、复合仿射变换等

1.3 凸函数的共轭函数
- 共轭函数的定义和性质
- 共轭函数的计算示例
- 共轭函数在优化中的应用

#### 第2章：凸优化问题（4学时）
2.1 凸优化问题的定义和标准形式
- 凸优化问题的一般形式
- 线性规划、二次规划、半定规划等标准问题
- 凸优化问题的性质：局部最优解即为全局最优解

2.2 凸优化问题的转化
- 不等式约束转化为等式约束
- 非凸问题的凸松弛
- 变量替换和问题重构

2.3 凸优化问题的示例
- 机器人路径规划中的凸优化模型
- 机器人控制中的凸优化问题

#### 第3章：凸优化的对偶理论（4学时）
3.1 拉格朗日对偶问题
- 拉格朗日函数的定义
- 对偶问题的构造和性质
- 弱对偶性和强对偶性

3.2 KKT条件
- KKT条件的推导和几何意义
- KKT条件的应用：最优性条件和互补松弛性
- KKT条件在机器人优化问题中的应用

3.3 对偶问题的求解
- 对偶上升法
- 次梯度方法
- 对偶分解方法

#### 第4章：凸优化算法（4学时）
4.1 无约束凸优化算法
- 梯度下降法：批量梯度下降、随机梯度下降
- 牛顿法和拟牛顿法
- 共轭梯度法

4.2 约束凸优化算法
- 投影梯度下降法
- 内点法
- 增广拉格朗日方法

4.3 算法收敛性分析
- 收敛速度的定义和分析方法
- 不同算法的收敛性能比较
- 算法参数的选择和调优

### 第二部分：多目标优化（16学时）

#### 第5章：多目标优化基础（4学时）
5.1 多目标优化问题的定义
- 多目标优化问题的一般形式
- 目标函数之间的冲突性
- 多目标优化与单目标优化的区别

5.2 帕累托最优性理论
- 帕累托最优解的定义
- 帕累托前沿的概念
- 帕累托最优解的性质

5.3 多目标优化的解的概念
- 弱帕累托最优解、强帕累托最优解
- 真帕累托最优解
- 帕累托最优解的存在性条件

#### 第6章：多目标优化算法（6学时）
6.1 标量化方法
- 加权和方法
- ε-约束方法
- 目标规划方法

6.2 进化多目标优化算法
- 遗传算法在多目标优化中的应用
- NSGA-II算法
- MOEA/D算法

6.3 多目标优化的其他方法
- 帕累托邻域法
- 多目标梯度下降法
- 多目标内点法

#### 第7章：多目标优化在机器人中的应用（6学时）
7.1 机器人路径规划中的多目标优化
- 路径长度、时间、能量消耗多目标优化
- 避障和路径平滑的多目标平衡
- 多机器人路径规划的多目标协调

7.2 机器人任务分配中的多目标优化
- 任务完成时间和能耗的多目标优化
- 多机器人任务分配的帕累托最优解
- 动态任务环境中的多目标决策

7.3 机器人控制中的多目标优化
- 跟踪精度和控制能量的多目标优化
- 机器人姿态控制的多目标设计
- 多机器人协调控制的多目标优化

### 第三部分：流形优化（16学时）

#### 第8章：流形的基本概念（4学时）
8.1 拓扑流形和微分流形
- 流形的定义和基本性质
- 流形的局部坐标系和坐标变换
- 流形的维数和定向

8.2 切空间和余切空间
- 切向量的定义和性质
- 切空间的基和坐标表示
- 余切空间和微分形式

8.3 流形上的向量场和张量场
- 向量场的定义和运算
- 张量场的概念和类型
- 黎曼度量和内积

#### 第9章：流形优化的基本理论（6学时）
9.1 流形上的优化问题
- 流形上的目标函数和约束
- 流形优化问题的定义
- 流形优化与欧几里得空间优化的区别

9.2 流形上的梯度和海森矩阵
- 黎曼梯度的定义和计算
- 黎曼海森矩阵的概念
- 流形上的泰勒展开

9.3 流形优化的最优性条件
- 流形上的一阶最优性条件
- 流形上的二阶最优性条件
- 流形优化问题的对偶理论

#### 第10章：流形优化算法（6学时）
10.1 流形上的梯度下降法
- 黎曼梯度下降法
- 步长选择和线搜索
- 收敛性分析

10.2 流形上的牛顿法
- 黎曼牛顿法的构造
- 海森矩阵的近似计算
- 牛顿法的收敛速度

10.3 其他流形优化算法
- 流形上的共轭梯度法
- 流形上的拟牛顿法
- 随机流形优化算法

#### 第11章：流形优化在机器人中的应用（6学时）
11.1 机器人姿态估计中的流形优化
- 特殊正交群SO(3)上的优化
- 李群流形上的姿态估计
- 多传感器融合中的流形优化

11.2 机器人运动规划中的流形优化
- 构型空间流形上的路径规划
- 流形上的最优控制问题
- 冗余机器人的运动学优化

11.3 机器人视觉中的流形优化
- 相机标定中的流形优化
- 视觉SLAM中的流形优化
- 三维重建中的流形优化

## 实验环节
1. **实验一**：凸优化基础算法实现（8学时）
   - 实现梯度下降法和牛顿法
   - 求解机器人路径规划的凸优化问题
   - 分析算法收敛性能

2. **实验二**：多目标优化算法实现（8学时）
   - 实现NSGA-II算法
   - 求解多机器人任务分配问题
   - 分析帕累托前沿的分布特性

3. **实验三**：流形优化算法实现（8学时）
   - 实现SO(3)流形上的梯度下降法
   - 求解机器人姿态估计问题
   - 比较流形优化与欧几里得优化的性能
