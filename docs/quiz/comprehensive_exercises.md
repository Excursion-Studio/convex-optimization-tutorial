# 综合题

## 大题1：生产计划优化

**问题描述**：某工厂生产两种产品A和B，生产每单位产品A需要消耗2单位原料X和1单位原料Y，利润为3元；生产每单位产品B需要消耗1单位原料X和2单位原料Y，利润为4元。工厂每天可获得原料X的供应量为100单位，原料Y的供应量为80单位。

**要求**：
1. 建立线性规划模型，求解最大利润和对应的生产计划。
2. 使用单纯形法求解该问题。
3. 使用对偶理论求解该问题。
4. 使用CVXPY库求解该问题，并验证结果的一致性。

## 大题2：投资组合优化

**问题描述**：投资者有100万元资金，计划投资于三种资产：股票A、股票B和债券C。根据历史数据，股票A的预期收益率为10%，标准差为20%；股票B的预期收益率为8%，标准差为15%；债券C的预期收益率为5%，标准差为5%。股票A和股票B的相关系数为0.6，股票A和债券C的相关系数为-0.2，股票B和债券C的相关系数为-0.1。

**要求**：
1. 建立均值-方差投资组合优化模型，求解最小方差投资组合。
2. 建立均值-方差投资组合优化模型，求解给定预期收益率为7%时的最小方差投资组合。
3. 使用拉格朗日乘数法求解上述问题。
4. 使用CVXPY库求解上述问题，并验证结果的一致性。

## 大题4：机器学习中的正则化优化

**问题描述**：在机器学习中，线性回归模型的正则化是一种常用的防止过拟合的方法。考虑以下带有L1正则化（LASSO）和L2正则化（岭回归）的线性回归问题：

$$\min_{w} \frac{1}{n} \sum_{i=1}^n (y_i - w^T x_i)^2 + \lambda \Omega(w)$$

其中，$\Omega(w)$ 是正则化项，$\lambda$ 是正则化参数。

**要求**：
1. 当 $\Omega(w) = \lVert w \rVert_2^2$ 时（岭回归），推导最优解的闭式表达式。
2. 当 $\Omega(w) = \lVert w \rVert_1$ 时（LASSO），解释为什么没有闭式表达式，并简述如何求解。
3. 使用梯度下降法求解岭回归问题。
4. 使用CVXPY库求解LASSO和岭回归问题，并比较不同正则化参数对结果的影响。

## 大题5：非线性优化问题

**问题描述**：求解下列非线性优化问题：

$$\min_{x} x_1^4 + 2x_2^4 + 3x_3^4$$

$$\text{s.t. } x_1 + x_2 + x_3 = 1$$

$$x_1, x_2, x_3 \geq 0$$

**要求**：
1. 使用拉格朗日乘数法求解该问题。
2. 使用梯度下降法求解该问题。
3. 使用牛顿法求解该问题。
4. 比较不同方法的收敛速度和求解精度。
